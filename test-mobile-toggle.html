Mobile Menu Toggle Test

This page tests the mobile menu toggle functionality.

Test Instructions

  1. Resize your browser window to less than 980px width (or use mobile device)
  2. Click the mobile menu toggle button (hamburger icon)
  3. Verify the mobile navigation menu expands
  4. Check that the toggle button icon animates to an "X"
  5. Click the toggle button again
  6. Verify the mobile navigation menu collapses
  7. Check that the toggle button icon returns to hamburger

Expected Behavior

  • Toggle button: Should have is-open class when menu is open
  • Mobile nav: Should have is-open class when menu is open
  • Body element: Should have menu-open class when menu is open
  • aria-expanded: Should be "true" when open, "false" when closed

Manual Verification

Open browser DevTools and inspect the following elements when toggling:

  • <button class="mobile-toggle"> - Check for is-open class
  • <nav class="mobile-nav"> - Check for is-open class
  • <body> - Check for menu-open class
  • <button aria-expanded="..."> - Check attribute value
